# Picqer Stats for Home Assistant

Create sensors for the following stats. (https://picqer.com/en/api/stats)

- Open Picklists
- Open Orders
- New Orders Today
- New Orders This Week
- Closed Picklists This Week
- Total Orders
- Backorders
- Closed Picklists Today
- New Customers This Week
- Total Products
- Active Products
- Inactive Products
